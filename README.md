📌 서비스 개요

방문 전 QR 길잡이는 근로복지공단 지사를 방문하기 전, 민원 유형별로 필요한 준비서류, 자가 확인사항, 공식 신청 경로를 미리 확인할 수 있도록 만든 모바일 최적화 웹 서비스입니다.

방문자가 QR을 스캔하면 별도 앱 설치 없이 웹페이지에 접속하여 본인 상황에 맞는 민원 유형을 확인하고, 필요한 서류와 온라인 신청 가능 경로를 사전에 점검할 수 있습니다.

본 서비스는 민원인의 방문 준비를 돕기 위한 비공식 안내 도우미이며, 실제 신청 가능 여부와 처리 결과는 반드시 근로복지공단 공식 안내 및 담당자 확인을 통해 최종 확인해야 합니다.

🎯 개발 목적

이 프로젝트는 방문 민원인이 지사를 방문하기 전에 필요한 정보를 쉽게 확인할 수 있도록 돕는 것을 목적으로 합니다.

주요 목적은 다음과 같습니다.

- 방문 전 준비서류 누락 방지
- 민원 유형별 신청 대상과 절차 사전 안내
- 온라인 신청 가능 업무의 공식 경로 연결
- 지사 방문 전 자가 체크를 통한 상담 효율 향상
- 개인정보를 수집하지 않는 정적 안내 서비스 구현
- 산재보상·재활 안내 서비스와 연계한 확장형 QR 안내 체계 구축

- 이름 입력 없음
- 주민등록번호 입력 없음

- 계좌번호 입력 없음

- 로그인 없음
